# digisfera-llm-api

Step 1. Add the repository

```
	<repositories>
		<repository>
		    <id>jitpack.io</id>
		    <url>https://jitpack.io</url>
		</repository>
	</repositories>

```

Step 2. Add the dependency

```
	<dependency>
	    <groupId>it.digisfera</groupId>
	    <artifactId>digisfera-llm-ollama</artifactId>
	    <version>1.0.0</version>
	</dependency>
```
